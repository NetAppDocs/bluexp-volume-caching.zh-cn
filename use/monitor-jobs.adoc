= 在时间线上监控卷缓存作业
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以监控所有卷缓存作业并确定其进度。

.步骤
. 从顶部菜单中、选择 image:settings-icon.png["UI上的设置图标"] >*时间线*。
+
image:timeline.png["时间线屏幕"]

. (可选)按时间、服务、操作、代理、 资源、用户或状态。
. 了解所有缓存和操作。
